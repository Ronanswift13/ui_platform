# 占位符数据目录 - MV_35kV / meter

## 说明
此目录用于存放现场采集的训练数据。

## 电压等级信息
- 名称: 35kV中压
- 类别: MV
- 热成像阈值: 正常<50°C, 预警<65°C, 报警>75°C

## 检测类别 (3个)
- 0: digital_meter
- 1: analog_meter
- 2: transformer_monitor

## 数据要求
1. 图像格式: JPG/PNG, 建议分辨率 1920x1080 或更高
2. 标注格式: YOLO格式 (class_id x_center y_center width height)
3. 建议样本数: 每类至少100张图像

## 采集建议
- 多角度拍摄设备
- 包含正常和异常状态
- 不同光照条件 (白天/夜间)
- 不同天气条件 (晴天/阴天/雨天)
